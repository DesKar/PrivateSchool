In order to correct this assignment:

1. Please run the "1.CREATE_TABLES.sql" in Workbench to create the tables.
2. Then, run the "2.INSERT_DATA.sql" in Wokrbench to populate the tables.
3. Please make sure to add the root username and password in "Database.java" in order to be able to connect to the database.

NOTES
The required SQL queries are in "3.ASSIGNMENT_QUERIES.sql" and in the java source files.
The rest of the files are auxiliary files used to construct the queries that are then used in the java source files.
